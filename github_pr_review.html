Description

Scaffolds the Hybrid MQTT-Based Regional EEWS extending DL_Spectra (IDA-PTW, IEEE Access submitted April 2026) into a distributed, broker-mediated deployment for the Java-Sunda Subduction Zone (InaTEWS pilot). MQTT v5 is elevated from an implementation detail to a first-class research contribution.

Suggested change
def _bayesian_fuse(psa_by_sta, sigma_table):
    # Al Atik (2010): σ²_total = τ² + φ². Only φ averages across stations.
    P   = np.stack([p.log10_psa for p in psa_by_sta.values()])       # (N, 103)
    TAU = np.stack([sigma_table.lookup_tau(p) for p in psa_by_sta])  # (N, 103)
    PHI = np.stack([sigma_table.lookup_phi(s, p) for s, p in psa_by_sta.items()])
    W   = 1.0 / np.square(PHI)                                       # within-event precision
    mu  = (W * P).sum(axis=0) / W.sum(axis=0)
    phi_post  = np.sqrt(1.0 / W.sum(axis=0))
    sigma_post = np.sqrt(TAU[0] ** 2 + phi_post ** 2)                # τ does NOT shrink by √N
    return mu.tolist(), (mu - sigma_post).tolist(), (mu + sigma_post).tolist()
Ref: Al Atik et al., SRL 81(5), 2010.

Alternative A · Replace MQTT with NATS JetStream

"""Alternative bus using NATS JetStream.
Rejected in ADR-0001 discussion: stronger durability guarantees but far fewer

import asyncio
from nats.aio.client import Client as NATS
from nats.js.api import StreamConfig

Why rejected: (i) no Pydantic/paho-equivalent Python ecosystem for edge agents; (ii) NATS retention duplicates the planned SeedLink archival mirror; (iii) added operational burden of JetStream clustering for a feature we don't yet need.
