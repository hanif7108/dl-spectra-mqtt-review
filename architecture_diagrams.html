6 · Latency budget

Wall-clock from P-onset to first published alert in milliseconds. Stage 0 alone reaches 630 ms for near-field; full Stage 2 chain stays under 3.78 s for the 3 s PTW path.

gantt
  title First-alert latency at PTW 3 s
  dateFormat  X
  axisFormat  %L
  section Ingest
  SeedLink to broker          :done, a1, 0,   50
  Broker fan-out              :done, a2, 50,  70
  section Stage 0 Near-field
  URPD 0.5 s window + infer   :crit, b1, 70,  150
  section Feature chain
  Features rolling buffer     :     c1, 150, 300
  Gate Stage 1 XGBoost        :     c2, 300, 330
  section Stage 2 Spectrum
  DLUHS2 fold ensemble CPU    :active, d1, 330, 730
  section Publish
  Fusion + alert publish      :     e1, 730, 780

Total overhead ≤ 780 ms; add the PTW window (3 000 · 5 000 · 8 000 ms) for the full wall-clock alert time. The sub-second URPD near-field flag reaches consumers at t ≈ 630 ms.
